# Mon Budget — Application Android

Application de gestion de budget personnel avec suivi des dépenses, revenus récurrents et alertes.

## Fonctionnalités
- Tableau de bord avec budget restant
- Suivi des dépenses par catégorie
- Charges et revenus récurrents automatiques
- Économies mensuelles et cumul annuel
- Alertes à 50%, 80% et 100% du budget

## Build
Le fichier APK est généré automatiquement via GitHub Actions à chaque push.
Téléchargez l'APK dans l'onglet **Actions** → dernier workflow → **Artifacts**.
